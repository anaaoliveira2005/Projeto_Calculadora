# Projeto Calculadora

Projeto desenvolvido em Java com o objetivo de aplicar conceitos de refatoração, testes funcionais e documentação utilizando JavaDoc.

## Objetivo

Aplicar técnicas de revisão estática, refatoração de código, tratamento de exceções e documentação JavaDoc em um projeto Java.

## Tecnologias Utilizadas

- Java
- VSCode
- Git
- GitHub
- JavaDoc

## Operações Implementadas

- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento para divisão por zero
- Tratamento para operadores inválidos
